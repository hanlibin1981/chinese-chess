# 中国象棋游戏 - 规格说明书

## 1. 项目概述

- **项目名称**: Chinese Chess Arena (中国象棋竞技场)
- **项目类型**: JavaWeb Web应用
- **核心功能**: 带用户系统、棋谱保存、观战悔棋的在线中国象棋对弈平台
- **目标用户**: 象棋爱好者

## 2. 技术栈

- **后端**: Spring Boot 3.2 + MyBatis + MySQL
- **前端**: Thymeleaf + Bootstrap 5 + 原生JS
- **WebSocket**: 实时观战、即时通信

## 3. 功能模块

### 3.1 用户系统
- 用户注册 (用户名、密码)
- 用户登录/登出
- 用户信息展示 (胜率、总局数)

### 3.2 对战模块
- **人人对战**: 双方通过WebSocket实时对弈
- **人机对战**: 简单AI (评估函数 + Alpha-Beta剪枝)
- 棋盘显示、落子校验
- 悔棋功能 (请求对方同意)

### 3.3 棋谱模块
- 保存棋谱 (XML/JSON格式)
- 棋谱列表展示
- 棋谱复盘 (逐步回放)

### 3.4 观战模块
- 大厅显示进行中的对局
- 实时观战 (WebSocket推送)
- 观战列表

## 4. 数据库设计

### users
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| username | VARCHAR(50) | 用户名 (唯一) |
| password | VARCHAR(128) | 密码 (MD5) |
| created_at | DATETIME | 注册时间 |

### games
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| player_red | BIGINT | 红方用户ID |
| player_black | BIGINT | 黑方用户ID (AI时为NULL) |
| winner | VARCHAR(10) | 胜方 (red/black/draw) |
| pgn | TEXT | 棋谱 (JSON格式) |
| is_ai | BOOLEAN | 是否人机对战 |
| created_at | DATETIME | 开始时间 |
| ended_at | DATETIME | 结束时间 |

## 5. 页面结构

1. **首页** (`/`) - 大厅、正在进行的对局列表
2. **登录** (`/login`) - 登录表单
3. **注册** (`/register`) - 注册表单
4. **对局室** (`/game/{id}`) - 棋盘、对战界面
5. **棋谱库** (`/games`) - 历史棋谱列表
6. **复盘** (`/review/{id}`) - 棋谱回放

## 6. 象棋规则

- 遵循标准中国象棋规则
- 帅(将)不能出宫
- 士走斜线
- 象(相)走田字(不能过河)
- 马走日
- 车走直线
- 炮隔子吃子
- 兵(卒)过河可横走

## 7. AI设计

- 评估函数: 棋子分值 + 位置分值
- 搜索深度: 3层
- 算法: Alpha-Beta剪枝

- [ ] 可以创建人人对战房间
- [ ] 可以创建人机对战
- [ ] 落子规则正确校验
- [ ] 可以悔棋 (需对方同意)
- [ ] 可以保存棋谱
- [ ] 可以查看和复盘历史棋谱
- [ ] 可以观战正在进行的对局
