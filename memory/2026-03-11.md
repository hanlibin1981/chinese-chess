# 中国象棋游戏开发记录

## 2026-03-11

### 项目概述
开发一个中国象棋人机对战游戏，使用 Spring Boot + WebSocket + H2 数据库。

### 技术栈
- 后端: Spring Boot 3.2, Java 21
- 前端: Thymeleaf + SockJS + STOMP
- 数据库: H2 (嵌入式)
- AI: 简单随机走法

### 关键修复
1. **棋子命名**: 使用单个汉字 (車、馬、炮、兵等)，通过 isRed 数组追踪颜色
2. **棋子颜色**: ChessBoard 添加 boolean[][] isRed 数组，复制/移动时同步更新
3. **吃子逻辑**: isSameSide() 方法使用 isRed 数组判断同侧，解决过河兵/炮横向吃子问题
4. **数据库**: 使用 MERGE INTO 避免重复插入默认用户
5. **前端**: 修复 Thymeleaf 变量解析，添加 valid-move 高亮显示
6. **将军判断**: 暂时禁用，避免无限递归
